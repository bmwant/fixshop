Run application with vagrant
--------
1. vagrant up  # from this directory
2. vagrant ssh  # connect to virtual machine 
3. cd /vagrant    
	python3.4 manage.py runserver 0.0.0.0:8080
4. Server is running, go to
    192.168.33.10:8080
    and test web-application
